# Note Routing

Use this guide to choose the destination note/folder.

## Inbox
Use for fast capture when structure is unclear.

## Daily Notes
Use for same-day chronology, status snapshots, and activity traces.

## Sessions
Use for summaries of important conversations, work sessions, and outcome-focused notes.

## Projects
Use for active initiatives that have evolving state.
Store:
- goals
- status
- milestones
- linked decisions
- open questions

## Decisions
Use when a choice has rationale and future consequences.
Store:
- context
- decision
- why
- alternatives
- follow-up

## Knowledge
Use for evergreen facts, repeatable procedures, and distilled know-how.

## Entities
Use for stable information about a person, company, tool, service, or system.

## Logs
Use for incidents, debugging traces, experiments, and diagnostics.

## Archive
Use for completed or stale notes that should remain searchable.
